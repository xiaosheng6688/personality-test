# 🧠 人格探索 - 发现真实的自己

一个基于 MBTI 理论的人格测试网站，帮助用户了解自己的性格类型。

## ✨ 特色功能

- 🎯 **精准测评** - 基于心理学理论的科学测评体系
- 📊 **深度分析** - 多维度人格特质可视化展示
- 🤝 **社交匹配** - 发现与你最契合的灵魂伴侣
- 💾 **本地存储** - 所有数据仅存储在本地，保护隐私
- 📱 **响应式设计** - 完美适配手机和电脑

## 🚀 快速开始

```bash
# 本地预览
npm install
npm run dev

# 部署到 Vercel
npm run deploy
```

## 📋 人格类型

包含 16 种 MBTI 人格类型的完整分析：
ISTJ, ISFJ, INFJ, INTJ, ISTP, ISFP, INFP, INTP,
ESTP, ESFP, ENFP, ENTP, ESTJ, ESFJ, ENFJ, ENTJ

## 🛠 技术栈

- HTML5 + CSS3 + JavaScript (Vanilla)
- 纯前端实现，无需后端
- 支持部署到任何静态托管平台
